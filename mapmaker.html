How to save a map:
1. Edit tiles, portals, Migs
2. Click ⬇ Download All JSONs
3. Upload the .json files to secretsoflerma/zones/ (frontend) and secretsoflerma-server/zones/ (backend)
4. Upload zones_export.js → replace zones.js on server
5. Restart server with pm2 restart all
